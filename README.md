# Cli Clock

## Compilation

- Linux : `cc -l curses` 

- FreeBSD : `cc -I/usr/local/include -L/usr/local/lib -l curses`

## Features

- Centered clock with seconds, 24h format

- ~~User can change fg color with -c or --color parameter~~ Broken?

- Use 'q' to quit
