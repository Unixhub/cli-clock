# Cli clock

- Compile with `cc -lcurses` (`cc -I/usr/local/include -L/usr/local/lib -lcurses` for FreeBSD)

- Centered clock with seconds, 24h format

- User can change fg color with -c or --color parameter

- Use 'q' to quit
